Hello, I'm Noah. These are side projects to help friends and to survey new technique.

Firstly, Online Judge System is the first side project during my master degree. It's created for TAs to check students' code efficiently. The website uses Apache server on Amazon AWS EC2. Its back-end is Python-cgi and use SQLite to handle the storage.

Then, Sale System (web) is born to help my parents who are not familiar to computers. In the past, they used physical calculator, papers and pens to write down each sales. At the end of month, they made handwritten sale invoices to customers. Why not to use computers? Because they're afraid of drawing a well-layouted document on Excel. But thanks to iPhone, they are not fear to web browsing. This side project uses Python simple-HTTP server to provide localhost service.
